# hospital-emergency-department
Semester long group project. (January 2026 - May 2026)

We created patient data that was separated into tables of patients (general information), procedures (procedure linked to each patient), pt_history (date of procedure), and procedure_list (list of all procedures available). The CSV files were uploaded to MySQL, which was then linked to Java.

In Java, we created the Emergency Department menu. The user is able to view every table. You can also update every field on each table. Delete allows the user to delete a whole row from a table, while Add allows the user to create a new patient.

At the current stage, our project is fully functional. All files perform their intended task successfully.
